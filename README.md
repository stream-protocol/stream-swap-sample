# StreamSwap sample

A swap sample using necessary components and functions from [Raydium frontend.](https://github.com/raydium-io/raydium-frontend).

## Install pkg

yarn

## Start service

yarn start

## Demo site

https://swap-sample.vercel.app/

## Warning

You should use it at your own risk.
